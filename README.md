# Study Buddy Project

A Python project for study assistance.

## Project Structure

```
study_buddy-proj/
├── src/
│   ├── __init__.py
│   └── app.py
├── tests/
│   ├── __init__.py
│   └── test_app.py
├── main.py
├── Pipfile
├── requirements.txt
├── .gitignore
└── README.md
```

## Getting Started

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python main.py
   ```

## Running Tests

```
python -m pytest tests/
```
