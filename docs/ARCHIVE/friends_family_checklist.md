# StudyBuddyAI - Friends & Family Beta Testing Checklist

## מדריך לבודקי בטא (Beta Testers Guide)

ברוכים הבאים לבדיקת StudyBuddyAI! המסמך הזה יעזור לכם לבדוק את כל הפיצ'רים של האפליקציה.

---

## 1. דרישות מקדימות

### מכשירים לבדיקה
- [ ] מחשב שולחני / נייד
- [ ] טלפון נייד (אנדרואיד או אייפון)
- [ ] טאבלט (אם יש)

### דפדפנים לבדיקה
- [ ] Chrome
- [ ] Safari
- [ ] Firefox
- [ ] Edge

---

## 2. בדיקות בסיסיות

### 2.1 דף הבית
- [ ] האם הדף נטען ללא שגיאות?
- [ ] האם הטקסט בעברית מוצג מימין לשמאל?
- [ ] האם תמונת אבנר (הקפיברה) נטענת?
- [ ] האם הכפתורים עובדים?
- [ ] האם הניווט עובד בלחיצה על "בית" / "מסכם" / "כרטיסיות" וכו'?

### 2.2 תפריט ניווט
**במחשב:**
- [ ] האם התפריט העליון מוצג?
- [ ] האם ניתן לנווט לכל הכלים?

**במובייל:**
- [ ] האם כפתור "המבורגר" (≡) מוצג?
- [ ] האם לחיצה פותחת תפריט?
- [ ] האם ניתן לנווט מהתפריט?
- [ ] האם לחיצה על X סוגרת את התפריט?

---

## 3. בדיקת כלי המסכם

### 3.1 העלאת טקסט
1. [ ] לכו לעמוד "מסכם"
2. [ ] הדביקו טקסט בתיבה (פסקה מויקיפדיה למשל)
3. [ ] לחצו "צור סיכום"
4. [ ] האם מוצגת אנימציית טעינה?
5. [ ] האם התוצאה מופיעה?
6. [ ] האם הסיכום הגיוני?

### 3.2 העלאת קובץ
1. [ ] לחצו על "בחר קובץ"
2. [ ] העלו קובץ PDF או Word
3. [ ] לחצו "צור סיכום"
4. [ ] האם הקובץ עובד?
5. [ ] **נסו קובץ גדול מ-10MB** - האם מוצגת שגיאה?

### 3.3 שגיאות
- [ ] **נסו בלי טקסט** - האם מוצגת הודעת שגיאה?
- [ ] האם השגיאה בעברית וברורה?

---

## 4. בדיקת כרטיסיות

### 4.1 יצירת כרטיסיות
1. [ ] לכו לעמוד "כרטיסיות"
2. [ ] הדביקו טקסט (למשל הגדרות ממילון)
3. [ ] לחצו "צור כרטיסיות"
4. [ ] האם הכרטיסיות נוצרות?

### 4.2 שימוש בכרטיסיות
- [ ] לחצו על כרטיסייה - האם היא מתהפכת?
- [ ] האם רואים את השאלה בצד אחד והתשובה בצד השני?
- [ ] הפעילו "מצב ערבוב" - האם הסדר משתנה?

---

## 5. בדיקת "בחן אותי"

### 5.1 יצירת מבחן
1. [ ] לכו לעמוד "בחן אותי"
2. [ ] הדביקו טקסט לימודי
3. [ ] לחצו "צור שאלות"
4. [ ] האם שאלות אמריקאיות נוצרות?

### 5.2 מענה על שאלות
- [ ] סמנו תשובות
- [ ] לחצו "בדוק תשובות"
- [ ] האם מוצג ציון?
- [ ] האם אבנר מגיב לציון? (חוגג אם גבוה, מעודד אם נמוך)
- [ ] לחצו "נסו שוב" - האם הכל מתאפס?

---

## 6. בדיקת עוזר שיעורים

### 6.1 שאילת שאלה
1. [ ] לכו לעמוד "עוזר שיעורים"
2. [ ] כתבו בעיה (למשל: "איך פותרים משוואה ממעלה שנייה?")
3. [ ] לחצו "עזור לי עם השיעורים"
4. [ ] האם מופיע הסבר שלב-אחר-שלב?

### 6.2 בדיקת התשובה
- [ ] האם יש שלב 1 - הבנת השאלה?
- [ ] האם יש שלב 2 - פירוק למשימות?
- [ ] האם יש שלב 3 - טיפים?
- [ ] האם ההסבר מובן?

---

## 7. בדיקות ויזואליות

### 7.1 תמונות אבנר
- [ ] האם תמונות אבנר מופיעות בכל הדפים?
- [ ] האם יש תמונות שונות למצבים שונים (טעינה, שגיאה, הצלחה)?
- [ ] האם התמונות נראות טוב על מסכים שונים?

### 7.2 צבעים ועיצוב
- [ ] האם הצבעים נעימים לעין?
- [ ] האם הטקסט קריא?
- [ ] האם הכפתורים בולטים?
- [ ] האם יש אפקט ריחוף (hover) על כפתורים?

### 7.3 רספונסיביות
- [ ] **צמצמו את חלון הדפדפן** - האם העיצוב מתאים?
- [ ] האם הכרטיסים נערמים במובייל?
- [ ] האם אפשר לגלול בקלות?

---

## 8. בדיקות ביצועים

### 8.1 זמני טעינה
- [ ] האם הדף הראשי נטען תוך 3 שניות?
- [ ] האם יש אינדיקציה כשמחכים לתוצאות?

### 8.2 יציבות
- [ ] נסו להשתמש באפליקציה 10 דקות - האם היא יציבה?
- [ ] נסו לפתוח כמה כרטיסיות - האם עובד?
